Implementar uma função chamada cumsum que recebe uma lista de números e retorno ao soma cumulativa; isto é, a função deve produzir uma nova lista onde o i-ésimo elemento é a soma de todos os elementos até a lista original. Por exemplo, se a função recebe uma lista t = [1, 2, 3,10] deve dar como resultado [1,3,6,16]. Atenção que de cada vez que a função é chamada produz a lista resultado de uma só vez.
